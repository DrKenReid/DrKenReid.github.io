# kenreid.co.uk

Personal portfolio website for Ken Reid — Data Scientist, photographer, guitarist, and avid reader.

**Live site:** [kenreid.co.uk](https://www.kenreid.co.uk)

## Pages

- **Home** — Introduction and overview
- **Data Science** — Professional bio, CV, and collaboration links
- **Photography** — Gallery of 364 photos with lazy loading, Chrome themes, and achievements
- **Music** — Last.fm listening stats and guitar performances
- **Literature** — Reading statistics, Goodreads shelves, and book reviews
- **Contact** — Email and social links

## Tech Stack

- HTML5, CSS3, JavaScript (ES5)
- [Bootstrap 4](https://getbootstrap.com/)
- jQuery 2.2.4, Popper.js
- [Isotope](https://isotope.metafizzy.co/) (gallery layout)
- [Owl Carousel](https://owlcarousel2.github.io/OwlCarousel2/) (image carousels)
- [WOW.js](https://wowjs.uk/) (scroll animations)
- [Jarallax](https://github.com/nk-developer/jarallax) (parallax backgrounds)

## Deployment

Hosted on [GitHub Pages](https://pages.github.com/) with a custom domain via CNAME. Push to `main` to deploy.
